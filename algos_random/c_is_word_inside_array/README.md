# Description
Function that takes an array of strings and a word as a parameters and return the true or false whether word is found in an array. Word can contain wildcards

# Examples:
Input: = ["cat","bat","put","dog"]

- output: TRUE
  * "dog"
  * "*u*"
  * "*og"
  * "***"

- output: FALSE
  * "cog"
  * "*t*"
  * "*"
  * "**"
